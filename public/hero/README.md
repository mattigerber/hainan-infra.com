# Hero Media

Use this folder to configure the Hero section video and optional cover image.

Files:

- `cover.png` (or `cover.jpg`, `cover.webp`, etc.): hero teaser cover image.
- `video.mp4` (or `.webm`, `.ogg`, `.mov`, `.m4v`): hero popup video.

Notes:

- Hero uses local media files from this folder only (no embedded URL player).
- Cover selection prefers files named `cover.*`; otherwise the first image file is used.
- Video selection prefers files named `video.*`; otherwise the first supported video file is used.
